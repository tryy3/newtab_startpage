Breach NewTab_startpage
=========

Simple Startpage

![Alt text](http://i.imgur.com/zSCIbGr.png)

  - Favorite Links
  - Weather
  - Notepad



Version
----

0.1.0



Installation
--------------
Note: Before you install newtab_startpage configure newtab_startpage!

```sh
right click, click on "Configure Modules"

copy/paste

github:N00D13/newtab_startpage
```

Configure
---------

* Set new Links
* paste your Weather Code

```sh
"/controls/newtab/"

Change the "background.jpg" for the Background

change the link in "<iframe id="forecast_embed" type="text/html" frameborder="0" height="245" width="100%" src="http://forecast.io/embed/#lat=53.514967&lon=8.133128&name=Wilhelshaven%20Germany&units=si"> </iframe>" for new Weather Settings
```
